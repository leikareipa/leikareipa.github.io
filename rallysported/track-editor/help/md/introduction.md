# Introduction

[RallySportED](https://www.tarpeeksihyvaesoft.com/rallysported) is a contemporary modding tool for the MS-DOS racing game [Rally-Sport](https://web.archive.org/web/19970805142345/http://www.cs.tut.fi/~k140734/Rally-Sport/index.html).

RallySportED is &copy; 2016&ndash;2023 [Tarpeeksi Hyvae Soft](https://github.com/leikareipa). No affiliation with Rally-Sport creator Jukka Jäkälä.
