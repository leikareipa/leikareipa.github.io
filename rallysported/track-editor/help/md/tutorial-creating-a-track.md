# Farting up a sample track

The following video shows a quick throwing together of a sample track.

<dokki-video headerless src="NaSuve7f7Cc"></dokki-video>

The steps in the video can be summarized like so:

1. Start with Rally-Sport's [track #1](/rallysported/?track=demoa) as a base (using the "?track=" URL parameter). Alternatively, you could drag and drop the ZIP file of a  RallySportED project you've downloaded previously.
1. In the terrain editor, level the terrain to a height of 0 by hitting <key-input>L</key-input>.
2. In the tilemap editor, paint the ground a solid grass with <mouse-input>Left click</mouse-input>.
3. Back in the terrain editor, clear away some of the track props (trees, billboards, etc.) with <mouse-input>Shift + Left click</mouse-input> over the props.
4. Use <mouse-input>Middle click</mouse-input> to paint a gravel road.
5. Add some decorative props with <mouse-input>Shift + Left click</mouse-input>, and <mouse-input>Right click</mouse-input> over a prop to change its type.
6. Record the CPU opponent's driving line.
7. Adjust the terrain height with <mouse-input>Left click</mouse-input> and <mouse-input>Right click</mouse-input>, and more precisely with <key-input>Up</key-input> and <key-input>Down</key-input>.
8. Use terrain tile #0 to add water.
9. Use terrain tiles #248 and #249 to add bridges.
10. Paint some textures in the texture editor.
