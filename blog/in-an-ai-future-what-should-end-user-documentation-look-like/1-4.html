Beginner's Quick-Start Guide for JavaScript Software Renderer

This guide will walk you through the basics of using the JavaScript software renderer. Follow the steps below to set up a simple scene and render it onto a canvas.

Step 1: Import the Renderer

Add the renderer's script to your HTML file:

<script src="distributable/rngon.js"></script>

Step 2: Create an HTML Canvas Element

Add a canvas element to your HTML file and give it an id attribute:

<canvas id="canvas"></canvas>

Step 3: Create a Scene

In your JavaScript code, create a scene by defining a mesh with one or more n-gons:

const quad = Rngon.ngon([
    Rngon.vertex(-1, -1, 0),
    Rngon.vertex(1, -1, 0),
    Rngon.vertex(1, 1, 0),
    Rngon.vertex(-1, 1, 0)], {
        color: Rngon.color(255, 255, 0),
});

const mesh = Rngon.mesh([quad], {
    rotation: Rngon.vector(0, 0, 45)
});

In this example, we create a quad (a polygon with four vertices) and assign a color to it. Then, we create a mesh containing that quad and apply a 45-degree rotation around the Z-axis.

Step 4: Render the Scene

Now that we have a mesh, we can render the scene using the Rngon.render() function:

Rngon.render({
    target: "canvas",
    scene: [mesh],
    options: {
        cameraPosition: Rngon.vector(0, 0, -5),
    },
});

In this example, we specify the target as the id attribute of the canvas element we created earlier. We also set the camera position so that it's placed 5 units away from the mesh along the Z-axis.

Step 5: Run Your Code

Save your HTML and JavaScript files, and then open the HTML file in a web browser. You should see the rendered scene on the canvas element.

Sample Complete Code

Here's a complete example of an HTML file that uses the renderer:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Software Renderer Quick Start</title>
    <script src="distributable/rngon.js"></script>
</head>
<body>
    <canvas id="canvas"></canvas>
    <script>
        const quad = Rngon.ngon([
            Rngon.vertex(-1, -1, 0),
            Rngon.vertex(1, -1, 0),
            Rngon.vertex(1, 1, 0),
            Rngon.vertex(-1, 1, 0)], {
                color: Rngon.color(255, 255, 0),
        });

        const mesh = Rngon.mesh([quad], {
            rotation: Rngon.vector(0, 0, 45)
        });

        Rngon.render({
            target: "canvas",
            scene: [mesh],
            options: {
                cameraPosition: Rngon.vector(0, 0, -5),
            },
        });
    </script>
</body>
</html>

This basic example should help you get started with the JavaScript software renderer. Refer to the API documentation for more information on available features and options.
