std::function<void(FILTER_FUNC_PARAMS)> filter_c::filter_metadata_s::apply

A reference to a relevant filter function provided by the filter functions interface, src/filter/filter_funcs.h, this function applies the filter's image processing to an image's pixels.

This function is expected to operate on the image's pixels in-place.

This function is allowed only to alter the image's pixel values, not e.g. its size.
