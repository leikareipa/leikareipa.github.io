System requirements
===================

- Windows 9x

- Some variant of the S3 ViRGE, with 4 MB of video memory

- Pentium 2 or better


Troubleshooting & Known issues
==============================

- The maximum supported resolution is 512 x 384 x 16. 400 x 300 x 32 or
  320 x 240 x 32 are recommended.

- Split screen doesn't work properly.

- Various minor graphical glitches.
