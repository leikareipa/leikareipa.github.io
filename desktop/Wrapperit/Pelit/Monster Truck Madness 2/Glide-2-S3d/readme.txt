System requirements
===================

- An OS supported by Monster Truck Madness 2

- S3d compatible video card (e.g. S3 ViRGE)

- Pentium 2 or better


Troubleshooting & Known issues
==============================

- Disable reflections and lens flare to avoid most visual glitches.

- For greater texture detail, use the VideoMode INI option to reduce the game's
  resolution and/or color depth; or select a video card with more VRAM.

- 2 MB cards are not able to run the game in resolutions higher than 512 x 384.

- Headlights and snow don't work properly.

- Foggy weather features more prominent object popin.

- Some other minor rendering issues.
