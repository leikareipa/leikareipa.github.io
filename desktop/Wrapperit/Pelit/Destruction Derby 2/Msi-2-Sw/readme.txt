System requirements
===================

- The Mystique version of the game

- An OS supported by the game

- A relatively fast CPU with SSE support


Usage
=====

Put the DLL and its corresponding INI file next to DD2.EXE. The INI file
contains additional settings for you to tweak.

The "FastRenderer" INI option requires support for 16-bit video modes and
RGB565 display surfaces.
